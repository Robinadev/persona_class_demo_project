TO DEPLOY TO VERCEL - FOLLOW THESE 3 SIMPLE STEPS:
════════════════════════════════════════════════════════════════════════════

STEP 1: PUSH YOUR CODE TO GITHUB (2 minutes)
─────────────────────────────────────────────────────────────────────────────

In your terminal:
  $ git add -A
  $ git commit -m "Deploy Talaritel to Vercel"
  $ git push origin main

✅ Your code is now on GitHub


STEP 2: SET UP DATABASE IN SUPABASE (3 minutes)
─────────────────────────────────────────────────────────────────────────────

Go to: https://app.supabase.com

1. Select your project
2. Click "SQL Editor"
3. Run these 3 SQL scripts (in order):
   
   • /scripts/01-create-tables.sql
   • /scripts/02-create-indexes.sql
   • /scripts/03-enable-rls.sql

✅ Your database tables are ready


STEP 3: DEPLOY TO VERCEL (3 minutes)
─────────────────────────────────────────────────────────────────────────────

Go to: https://vercel.com/dashboard

1. Click "+ New Project"
2. Click "Import GitHub Repository"
3. Search and select: personal_demo_class_project
4. Click "Import"
5. Scroll to "Environment Variables"
6. Add these 3 variables:

   NEXT_PUBLIC_SUPABASE_URL
   [Get from: Supabase > Settings > API > URL]
   
   NEXT_PUBLIC_SUPABASE_ANON_KEY
   [Get from: Supabase > Settings > API > Anon Key]
   
   SUPABASE_SERVICE_ROLE_KEY
   [Get from: Supabase > Settings > API > Service Role Key]

7. Click "Deploy"
8. Wait 2-5 minutes for deployment
9. Click the domain link ✅

YOUR APP IS NOW LIVE!
