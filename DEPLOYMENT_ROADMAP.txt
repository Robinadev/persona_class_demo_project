    │ → Select your project                                              │
    │ → Click "SQL Editor"                                               │
    └─────────────────────────────────────────────────────────────────────┘

    Step 2b: Run Migration Scripts (3 separate queries)
    ┌─────────────────────────────────────────────────────────────────────┐

    │     → New Query → Paste → Run                                      │
    │     ✓ Creates all 10 database tables                               │
    │                                                                     │
    │ 2️⃣  Copy: /scripts/02-create-indexes.sql                           │
    │     → New Query → Paste → Run                                      │
    │     ✓ Adds performance indexes                                     │
    │                                                                     │
    │ 3️⃣  Copy: /scripts/03-enable-rls.sql                               │
    │     → New Query → Paste → Run                                      │
    │     ✓ Enables Row Level Security                                   │
    │                                                                     │
    │ Expected: "Success" message after each ✓                           │
    │                                                                     │
    └─────────────────────────────────────────────────────────────────────┘

PHASE 4: DEPLOYMENT (2 minutes)
────────────────────────────────────────────────────────────────────────────────

    Step 4a: Push to GitHub
    ┌─────────────────────────────────────────────────────────────────────┐
    │ git add .                                                           │
    │ git commit -m "Talaritel: deployment ready"                        │
    │ git push origin main                                               │
    │                                                                     │
    │ This syncs your code with GitHub                                   │
    └─────────────────────────────────────────────────────────────────────┘

    Step 4b: Deploy to Vercel
    ┌─────────────────────────────────────────────────────────────────────┐
    │ 1. Go to: https://vercel.com/dashboard                             │
    │ 2. Click "Add New..." → "Project"                                  │
    │ 3. Select your GitHub repository                                   │
    │ 4. Click "Deploy"                                                  │
    │                                                                     │
    │ Wait for: ✓ Build succeeded                                        │
    │          ✓ Deployment complete                                    │
    │          ✓ Domains assigned                                        │
    │                                                                     │
    │ Your app is LIVE! 🎉                                               │
    │                                                                     │
    └─────────────────────────────────────────────────────────────────────┘
